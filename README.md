# D&D Initiative Tracker

My project is a web app that connects a Discord bot to a website for tracking combat in Dungeons and Dragons. Messages are sent in Discord and are added to the HTML using Node and Javascript

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install the necessary files
2. Make the Discord bot

Instructions for using the software:

1. Open Discord, the website, and your IDE (I used VSCode)
2. Start the server by running 'node server.js' in the terminal
3. Put input into the initiative chat in Discord
4. See all inputs on the website

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* node.js
* discord.js
